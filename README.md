# NVIDIA Brand New Websites

## Setup

1. Duplikat file `env` menjadi `.env`, konfigurasikan settings sesuai kebutuhan

2. Migrasi dan seed database dengan command
```bash
php spark migrate && php spark db:seed DevSeeder
```

3. Jalankan server!
```bash
php spark serve
```
